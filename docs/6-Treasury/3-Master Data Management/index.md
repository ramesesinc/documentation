<b>Master Data Management</b>
===

<br />

<b>Required Role: TREASURY.MASTER, TREASURY.AFO</b>

<br />

Before a collection can be operational, the master files need to be set up.

<br />
<br />
<br />

## <b>AF Types</b>

<br />

The Accountable Form (AF) Types - supports standard accountable forms.

<br />
<br />

1. To view AF Types, go to Main Menu > Master > Treasury > AF Types.

<br />

![alt text][af types]

<br />
<br />

2. To add AF type, click the <b>New Document</b> icon, fill up the required data and click the <b>Save</b> icon.

<br />

![alt text][new af type]

<br />
<br />

3. To open AF type, select an AF type from the list and click the <b>Open</b> icon or simply "double-click" the selected AF Type.

<br />

![alt text][open af type]

<br />
<br />

4. To edit AF type, select and open an AF type. Click the <b>Edit</b> icon, then click the <b>Save</b> icon after editing.

*Note: System created file cannot be edited*

<br />

![alt text][edit af type]

<br />
<br />

5. To remove AF type, click the <b>Remove</b> icon. A confirmation dialog box will pop up, click "yes" to proceed.

*Note: System created file cannot be removed*

<br />

![alt text][remove af type]

<br />
<br />

6. To print the AF Types list, click the <b>Print</b> icon. The print preview sheet will be displayed. Click again the  <b>Print</b> icon and the print dialog box will pop up.

<br />

![alt text][print af type]

<br />

![alt text][print2 af type]

<br />
<br />
<br />

## <b>Collection Types</b>

<br />

Collections are classified by types. Some LGU create reports based on collection type. Examples of collection type are General Collection and Business Permits. This also determines the type of screen handler it will use during collection. That means if one collects the business permit, a special screen for collecting business permit will be used. 

<br />
<br />
<br />

1. To view Collection Types, go to Main Menu > Master > Treasury > Collection Types. Collection types are categorized into: Draft, Active and Inactive.

<br />

![alt text][collection types]

<br />
<br />

2. To add Collection Type, click the <b>New Document</b> icon, fill up the required data and click the <b>Save</b> icon.

<br />

![alt text][new collection types]

<br />
<br />

3. To open Collection Types, select a Collection Type from the list and click the <b>Open</b> icon or simply "double-click" the selected Collection Type.

<br />

![alt text][open collection types]

<br />
<br />

4. To print the Collection Types list, click the  <b>Print</b> icon. The print preview sheet will be displayed. Click again the  <b>Print</b> icon and the print dialog box will pop up.

<br />

![alt text][print collection types]

<br />

## <b>Collection Groups</b>

<br />

Collection group is a facility used in the system to speed up collections for some transactions. Some collections that are not automated, for example, building permits contain several fees that are included. So that one does not have to individually add these fees, by choosing the collection group it automatically populates these fees saving the collector time. 

<br />
<br />
<br />

1. To view Collection Groups, go to Main Menu > Master > Treasury > Collection Groups.

<br />

![alt text][collection groups]

<br />
<br />

2. To add Collection Groups, click the <b>New Document</b> icon, fill up the required data and click the <b>Save</b> icon.

<br />

![alt text][new collection groups]

<br />
<br />

3. To open Collection Groups, select a Collection Group from the list and click the <b>Open</b> icon or simply "double-click" the selected Collection Group.

<br />

![alt text][open collection groups]

<br />
<br />

4. To print the Collection Groups list, click the  <b>Print</b> icon. The print preview sheet will be displayed. Click again the  <b>Print</b> icon and the print dialog box will pop up.

<br />

![alt text][print collection groups]
